Roles

There are a number of commands associated with the role plugin. These allow the administrator to promote, demote, or change user roles.

These commands are:
ban- bans the user for the specified amount of time, echoes a message with the ban reason to said user
softban - bans, then immediately unbans the user, removing everything they have posted in chat and echoing them the reason given for the ban
kick - kicks the user from the channel, echoes them the reason given
mute - mutes the user for given number of time in minutes, echoes the reason given to them
addrole - assigns a new given role to the given user
delrole - removes a given role from the given user
role - lists the roles that belong to the given user
Friends List Management

NightWatch keeps a custom friend list for each user in the server. Users are able to view and modify their friend list with a number of commands.

These commands are:
add - sends a friend request to a specified user
accept - accepts the friend request of a specified user
decline - refuses the friend request of the specified user
delete - deletes specified user from friend list
list - lists users friends (or your own if no field is given)
incoming/outgoing - lists all current requests (defaults to incoming)
Music and Playlist Management

NightWatch allows users to create and manage music playlists to play through the Discord application. Users can create queues of songs from a number of sources: Youtube, Spotify, or Soundcloud. The bot plays music by simply following the hyperlink provided to it by the user. This allows for multiple sources of music to play with no conflicts, including Youtube, Spotify, Soundcloud, etc.

The commands associated with playlist management are:
queue - adds the song at the link to the queue (if there is no queue the song begins immediately)
next - skips to the next song in the queue
back - plays the previous song in the queue
clear - clears the queue
pause - pauses the queue
resume - resumes the queue
